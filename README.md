### alexa-scrooge

``` bash
git clone https://github.com/hursey013/alexa-scrooge.git

# Install dependencies
bundle

# Start the sample app using your API keys available from the Plaid Dashboard:
# https://dashboard.plaid.com
PLAID_CLIENT_ID=[CLIENT_ID] \
PLAID_SECRET=[SECRET] \
PLAID_PUBLIC_KEY=[PUBLIC_KEY] \
ACCESS_TOKENS=[PUBLIC_KEY] \
ruby app.rb
# Go to http://localhost:4567
```
